# Django BookMyShow Clone 🎬

A movie ticket booking web application inspired by BookMyShow, built using Django.

## Features

- User authentication (Login/Register)
- Movie listings
- Theater selection
- Seat booking system
- Booked seat validation
- User profile bookings
- Responsive modern UI
- Dark cinematic theme

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

1. Clone repository

```bash
git clone <your-repo-url>
```

2. Create virtual environment

```bash
python -m venv venv
```

3. Activate virtual environment

```bash
venv\Scripts\activate
```

4. Install dependencies

```bash
pip install -r requirements.txt
```

5. Run migrations

```bash
python manage.py migrate
```

6. Start server

```bash
python manage.py runserver
```
